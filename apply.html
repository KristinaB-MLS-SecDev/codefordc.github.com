Leadership Positions

Join our leadership team!

Some of you are already stepping up as leaders in Code for DC. Some of you have good ideas or skills that could help us improve. We’d like to recognize your contributions while building a more sustainable, successful organization.

These are the positions we feel are important to running a successful brigade. Please apply even if you meet just a few of the qualifications. We can teach you the parts you don’t know. Or we can split up the role with someone else. These are not set in stone. What’s most important is your commitment to Code for DC, our work, our mission, and our code of conduct.

Questions? Ideas? Please feel free to reach out to Code for DC leadership. We’d love to help.

Leadership positions:

  • Delivery Lead
  • Member Community Organizer
  • Open Source Community Organizer
  • Storyteller/Outreach Coordinator
  • Training Coordinator
  • Community Data Wrangler/Open Data Evangelist
  • User Experience/Usability Coordinator
  • Project Lead

In addition to their individual duties, Code for DC leadership also:

  • Help organize and run hack nights
  • Know and enforce the code of conduct
  • Represent and promote Code for DC at events and meetups
  • Meet monthly with Code for DC co-captains
